# Test-specific TAT & Disruption Model (GitHub Pages)

This version is designed for the TrakCare **vertical extract**.

Expected columns:
- Episode Number
- Collection Date / Collection Time
- Registration Date / Registration Time
- Authorised Date / Authorised Time
- Hospital
- Ward
- Test Set Description

Behaviour:
- Uses **SA** rows only
- Excludes **SAM** rows automatically
- Lets you filter by test and optional ward/hospital text
- Shows observed baseline TAT and scenario model outputs

Recommended GitHub upload:
- index.html
- style.css
- app.js

Do not upload the CSV if you do not want data in the repo. Just upload it in the browser when the site loads.
